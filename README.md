## 📘 What this is

howtouseai is a set of beginner-friendly guides for working with AI tools and common AI setups. It focuses on plain steps for tasks like AI coding, self-hosting, RAG, embeddings, recommendation systems, and simple AI workflows.

Use this repo if you want to:
- Learn AI tools without a deep technical background
- Follow clear steps instead of long theory
- Set up common AI projects on your own machine
- Understand how different AI parts work together

## 🖥️ What you need

This guide set works best on a Windows PC with:
- Windows 10 or Windows 11
- A stable internet connection
- Enough free disk space for downloads and project files
- A modern browser like Edge, Chrome, or Firefox

For some guides, you may also want:
- A GitHub account
- A text editor
- Basic access to the Windows command line
- A GPU for larger local AI tasks, if you plan to run heavier models
